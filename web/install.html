Install

From download to first agent in a few minutes.

Install the app, pair the Chrome extension, then deploy a starter agent or build your own browser worker from the hub.

curl -fsSL https://ctxant.com/install.sh | sh
macOS 11+ Chrome required Menu-bar app
1

Install the Mac app

Use the one-line installer or download the DMG manually if you prefer the drag-to-Applications route.

2

Finish the onboarding wizard

Create the hub bot, paste your AI key, and lock access to your Telegram user id.

3

Load the Chrome extension

Point Chrome at the bundled extension folder, then let the localhost pairing handshake connect it automatically.

What happens next

Your hub bot becomes the deployment console.

Once the app and extension are live, the hub bot is where you either launch a preset or build your own browser agent.

01

Paste the install command

The script downloads the latest DMG, copies CtxAnt into Applications, and launches it immediately.

02

Create your hub bot

Use BotFather to make a Telegram bot for the hub, then paste the token into the onboarding wizard.

03

Install the Chrome extension

Production installs should use the Chrome Web Store listing when it is live. Until then, the fallback path is chrome://extensions → developer mode → Load unpacked using the bundled extension/ folder.

04

Deploy or build

Open the hub bot, send /start, then either pick a starter or tap ➕ Build your own agent.

Troubleshooting

Most install issues are simple: missing extension reload, app not running, or wizard setup cut short.

The menu bar icon is missing

Notched MacBooks can hide menu-bar items. Shift icons left, or reach the local dashboard directly at http://127.0.0.1:8766/dashboard.

The extension says disconnected

Make sure the app is running, then quit and relaunch it. The extension pairs to the backend during startup.

The setup feels incomplete

If the wizard was dismissed too early, relaunch CtxAnt and complete the required fields again before testing the bot.
